⚙ C++ Engine

          
Every sort and search runs through a real C++ binary compiled with g++ -O2 -std=c++17. The engine records each comparison, swap and pointer movement as a JSON trace — no JavaScript simulation.

        

        

          
🎞 Step-by-step Traces

          
The visualizer replays the C++ execution trace one step at a time, showing exactly which elements are compared or swapped with real O(n) counter data.

        

        

          
📚 Algorithms Covered

          
Bubble · Selection · Insertion · Merge · Quick · Binary Search · Linked List operations · Stack operations. Each with pseudocode overlay and complexity cards.
